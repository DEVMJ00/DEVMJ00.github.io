## 🏗️ Architecture du Projet

Le site est conçu comme une **Single Page Application (SPA) statique** avec des sections dynamiques gérées par JavaScript.

```text

├── assets/             # Images, icônes, vidéos de démo
├── code/               # Pages détaillées des projets (ex: rename-me.html, spotnix.html)
├── JS/                 # Scripts principaux (scripts.js, easter.js)
├── CSS/                # Styles spécifiques (code.css, style.css)
├── index.html          # Page d'accueil principale
├── manifest.json       # Configuration PWA (Progressive Web App)
└── README.md           # Documentation du projet
```
### Points forts techniques
- **Performance :** Utilisation de `loading="lazy"` pour les images, préchargement des polices Google Fonts, et scripts chargés en `defer`.
- **SEO & Social :** Balises Open Graph et Twitter Cards configurées pour un partage optimal sur les réseaux sociaux.
- **Accessibilité :** Respect des attributs `aria-label`, `alt` descriptifs et navigation au clavier.
- **PWA Ready :** Manifest JSON inclus pour une installation possible sur mobile/desktop.
- **Design System :** Utilisation de variables CSS (`:root`) pour une maintenance facile des couleurs et typographies.
